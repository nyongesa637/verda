# Verda

> Codex-built litigation toolkits for human rights defenders.

Verda is the open-core MVP for a Codex-native legal platform: it turns a
defender's messy case file into a deployable, case-specific litigation
toolkit. Each toolkit is real software the lawyer can audit, modify, and
ship.

This repository is in active development. See `AGENTS.md` for the agent
operating contract and `docs/` for the architecture, security, and demo
notes.

## Status

Pre-alpha. Pinned to the Kenyan Article 22 / 23 constitutional petition
track. East-Africa expansion (Uganda, Tanzania) is on the roadmap.
